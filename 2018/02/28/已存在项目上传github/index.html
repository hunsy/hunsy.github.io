已存在项目上传github

发表于 2018-02-28 | 分类于 Git

1. 打开命令行终端，进入项目所在的目录，将项目初始化为Git项目

执行上述命令，会在当前目录生成.git隐藏文件夹

2. 将本地文件添加到本地的git仓库中

如果目录中存在 .gitignore文件，会按照已有规则过滤不需要的文件。如果不需要添加所以文件，可以将.换成具体文件的名称。

3. 将已添加的文件提交到本地仓库

4. 在github上创建一个新的仓库

5. 复制项目主页上的仓库地址

6. 在命令行终端将本地仓库与远程仓库关联

7. 提交代码到git仓库
